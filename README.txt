Project description:
kgoedicke.com/Surface-Subdivision

1. Compile the program with "make"
2. On the command line, type "./shade" followed by the bezier surface file and the subdivision parameter.
An example input is 
	./subdivide teapot.bez 0.2

User interaction:
arrow keys - rotates the scene
shift+arrow keys - translation in scene
'w' - toggles between wireframe and filled
's' - toggles between flat and smooth shading
'+' and '-' - zoom in and out
